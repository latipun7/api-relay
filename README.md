## Getting Started

Getting up and running is as easy as 1, 2, 3.

1. Make sure you have [NodeJS](https://nodejs.org/) and `yarn` or [npm](https://www.npmjs.com/) installed.
2. Install your dependencies

   ```bash
   cd path/to/api-relay
   yarn
   ```

3. Start your app

   ```bash
   yarn dev
   ```
